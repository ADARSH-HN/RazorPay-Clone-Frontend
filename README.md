# Razorpay Clone - Frontend

A frontend recreation of Razorpay's landing page built using HTML and Tailwind CSS.  
This project focuses on replicating modern UI layouts, reusable sections, animations, and responsive design patterns.

## 🚀 Live Demo

Check out the hosted version here:

[Razorpay Clone](https://adarsh-hn.github.io/RazorPay-Clone-Frontend/)

## 📸 Preview

![preview](./assets/preview.png)


## ✨ Features

- Razorpay-inspired landing page UI
- Modern responsive layout structure
- Hero section with styled components
- Feature and information sections
- Infinite scrolling animations
- Hover effects and smooth transitions
- Utility-first styling using Tailwind CSS

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
